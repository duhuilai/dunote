# v0.1.40

## 修复：智能表格列设置菜单被遮挡
- **根因**：列设置菜单使用 `position: absolute` 渲染在表格 `<th>` 内部，而表格父容器有 `overflow: auto` + 外层 `overflow: hidden`，导致菜单被裁剪/遮挡。
- **修复**：列设置菜单与选项编辑器改为 `position: fixed`，基于触发按钮的视口坐标定位，脱离表格 overflow 限制，不再被遮挡。

## 新增：智能表格字段类型「进度条」
- 列类型新增「进度」，单元格显示为蓝色进度条，并支持输入 0–100 的百分比数值。
- 进度列参与底部统计栏的合计 / 均值计算。
- 排序时进度按数值比较。

## 修复：人员管理「修改」按钮无反应
- 之前人员列表的「编辑」按钮没有绑定 onClick，点击无反应。
- 修复：点击编辑后打开编辑弹窗，回填当前人员信息，保存时调用 `updatePerson` 并持久化到磁盘。

## 修复：任务管理新建任务后无法修改
- 任务详情弹窗原仅支持查看 + 完成，没有编辑入口。
- 修复：详情弹窗顶部新增「编辑」按钮，进入编辑模式后可修改任务名称、描述、负责人、参与人、起止时间、状态、进度；保存后调用 `updateTask` 并持久化。
- 负责人 / 参与人仍从「人员管理」中勾选，便于按人统计。
