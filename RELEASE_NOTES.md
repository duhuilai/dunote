# v0.2.2

> 修复日期选择器在页面底部被截断的问题（智能表格日期列 + 全局日期选择器）。

## 一、修复日期选择器靠近底部被截断
- 触发元素（日期输入框/单元格）靠近页面底部时，日历弹出层原本固定在元素下方，会被窗口底边截断、无法完整查看与选择。
- 改为打开及切换「日/月/年」视图时，用 `useLayoutEffect` 实测弹层尺寸动态定位：
  - 视口下方剩余空间不足容纳弹层时，自动翻到触发元素**上方**显示；
  - 同时约束水平位置不超出右边界，避免贴边被裁切。
- 涉及文件：`src/components/table/DataTableView.tsx`（智能表格日期列）、`src/components/ui/DateField.tsx`（全局中文日期选择器）。
