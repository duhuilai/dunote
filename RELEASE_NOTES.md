# duNote v0.1.29

## 修复 · 导出空白 + 新增「在文件夹查看」

### 导出空白修复
- 根因：笔记列表右键导出时取 `note.content`，而本地文件笔记未打开过时 `content` 为空串，原回退读盘逻辑又受「内容为空才读盘」条件限制，导致写出空白文件。
- 修复：列表导出改为**本地文件始终读取磁盘最新内容**（不再信任可能为空的列表 content）；内容确为空时提示「该笔记暂无内容可导出」。
- 图片说明：当前图片为网络图片（`<img src="URL">`），不影响 Markdown/HTML 文字导出；PDF 导出已增加图片预加载为 base64，避免跨域导致图片区域空白。

### 在文件夹查看
- 笔记右键菜单新增「在文件夹查看」（仅本地文件）：点击用访达/资源管理器打开该笔记所在文件夹。
- 文件夹右键菜单新增「在文件夹查看」：点击用文件管理器打开该文件夹。
