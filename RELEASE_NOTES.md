# v0.1.33

## 导出修复
- **导出 PDF 没反应**：原实现用 `window.print()` 走系统打印对话框，在 macOS WebView 上会静默无响应。改为前端用 `html2pdf.js`（html2canvas + jsPDF）直接生成真正的 `.pdf` 文件，跨平台稳定可用。
- **导出 Word 点了保存没反应**：根因是异常被静默吞掉、且 macOS 写入用户所选路径可能受安全限制。新增应用内自定义中文「保存」弹窗（文件名输入 + 中文「保存」按钮，写入「下载」文件夹），并对所有导出路径加 `try/catch` + toast 提示，成功/失败都可见；Word 生成改用 `Packer.toArrayBuffer` 更稳健。
- **导出保存按钮中文化**：原生系统保存对话框的确认按钮由操作系统绘制、无法汉化，故改为应用内自定义「保存」弹窗（按钮中文「保存」），并保留「选择其他位置…」走原生对话框。导出格式标签汉化为 Markdown 文档 / HTML 网页 / Word 文档 / PDF 文档。

## 智能表格
- **单选列**：空值选项不再显示「（空）」，改为空白。
- **日期列**：用自定义中文日历替换原生 `input[type=date]`，月份与星期均为中文（存储值仍为 ISO `YYYY-MM-DD`）。
