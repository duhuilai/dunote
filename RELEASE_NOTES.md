# v0.1.30

## 修复
- **导出 PDF 空白**：弃用 html2pdf.js（html2canvas 在 Tauri WebView 下易截成空白页），改为系统打印对话框（window.print）方案。导出 PDF 时会弹出打印窗口，macOS 选左下角「PDF → 存储为 PDF」、Windows 选「Microsoft Print to PDF」即可，文字与图片都能可靠进入 PDF。
- **图片随软件更新丢失**：插入图片（工具栏按钮 / 斜杠命令「图片」）时，自动把图片下载并内嵌进笔记内容（base64），不再依赖外部地址（如 localhost 临时服务、图床链接）。从此更新软件或离线，图片都随笔记一起保存，不会丢失。
  - 说明：此前已经失效的图片地址（URL 已打不开）无法自动找回，需要对该图片重新插入一次；之后插入的图片都会自动内嵌保存。
