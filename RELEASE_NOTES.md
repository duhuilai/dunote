# duNote v0.1.27

## 重构 · 智能表格（彻底重做，替代 v0.1.25 半成品）
旧实现把「列类型」仅存为表格属性，从不改变单元格渲染，且依赖 TipTap 表格命令导致插入行/列失效、单选等类型无选项录入界面。本次重做为**独立 Tiptap 节点 + React 渲染视图**，所有交互在 React 内完成，稳定可靠。

### 字段类型（9 种，点击列头「类型徽章」或「⋯」设置）
- 文本 / 数字 / 日期 / 单选 / 多选 / 人员 / 勾选 / 链接 / 评分
- 单选、多选、人员支持**选项管理**（标签 + 颜色），人员可自由录入标签
- 切换列类型即时生效，单元格按类型渲染为对应控件（下拉/勾选/星级/日期选择器等）

### 单元格与行列
- 单元格按类型呈现：单选下拉(带颜色)、多选/人员标签、勾选框、评分星级、链接可点击打开
- 工具栏「+ 行」「+ 列」可靠插入；每行 hover 可删除；列设置里可删除列、改名
- 输入/选择直接写入文档并自动保存

### 排序 / 筛选 / 统计
- 排序：选列 + 升序/降序（数字按数值、日期按时间、其余按中文排序）
- 筛选：多条件（包含 / 等于 / 为空），按列分别设置
- 统计栏：可见行数、数字列合计与均值、单选/多选/人员各选项计数（含颜色）
- 排序与筛选为视图态，可一键「重置视图」

### 使用
- 输入「/」唤起命令菜单，选择「智能表格」插入（默认含 名称/状态(单选)/日期 三列）；普通「表格」仍为自由富文本表格。
- 数据随笔记内容（HTML）持久化，兼容现有 Gitee 备份结构。
