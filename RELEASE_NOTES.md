# v0.2.0

> 本次为 0.1.x 系列的功能汇总大版本。此前 0.1.0–0.1.43 的全部新增与修复均合并进本版本说明，0.1.x 各小版本标签已归档清理。

## 一、富文本编辑器
- 基于 TipTap v3 的富文本编辑：标题、加粗、斜体、下划线、删除线、行内代码、链接、引用、有序/无序列表、代码块。
- 文本样式：字体大小调节、文本颜色、高亮颜色、左/中/右对齐。
- 斜杠命令（SlashCommand）：输入 `/` 快速插入图片、表格、智能表格等元素。
- 图片插入改为**系统文件选择对话框**从本地选图，自动读字节内嵌为 base64 data URL（彻底避免重开丢失）；浏览器 dev 环境无 Tauri 时回退为地址输入框。
- 基础表格：支持列宽拖拽（固定表头布局）。

## 二、智能表格（DataTable）
- 自定义 TipTap atom 节点 + React NodeView 渲染，列/行数据以节点属性序列化，复制粘贴、撤销重做均安全。
- 列类型：文本、数字、日期、下拉、进度条等；支持排序、筛选、按列统计。
- 列宽可调、固定表头、删除列、列设置收进菜单（Popover 通过 `createPortal` 挂到 `document.body`，修复点击无反应问题）。
- 新增「进度条」字段类型（空值按 0 处理，统计按数值、排序按数值）。

## 三、笔记管理
- 双模式：内存普通笔记 + 本地文件笔记（扫描文件夹自动入库）。
- **笔记类型持久化**：本地根目录 `.dunote-meta.json` 记录「相对路径 → 笔记类型」，扫描时还原，重启不再变回普通笔记；右键「修改笔记类型」可改并持久化。
- 新建 / 重命名 / 删除笔记，右键「在文件夹查看」直接打开文件所在目录。
- 历史记录：本地历史 + Gitee 历史同步；恢复历史可即时刷新（同笔记也重载）。

## 四、任务 / 人员
- 任务：负责人（下拉选择人员）、参与人（多选 chip）、状态，详情弹窗可编辑。
- 人员：增删改、在职/离职状态；入职日期用中文日历（可手动录入 `YYYY-MM-DD`，日/月/年三级视图，今天/清除）。

## 五、导出
- 导出为 Markdown / HTML / Word（真正 .docx，图片内嵌、列表层级、标题不重复）/ PDF。
- 自定义中文「保存」弹窗写入「下载」文件夹，支持「选择其他位置」走原生对话框。
- 修复：Word 颜色解析报错、PDF 离屏空白、Markdown 混合表头表格转换。

## 六、设置与同步
- 设置持久化（更新后不丢失）。
- Gitee 历史同步：每次生成历史向 `dunote-history/` 写 JSON 快照，base64 无损往返。
- **加固**：单文件 base64 超过 900KB 时返回明确中文报错，避免超限静默失败、恢复时找不到历史。

## 七、笔记内检索定位（新增）
- 工具栏「检索」按钮，或 `Ctrl+F` / `Cmd+F` 打开检索栏（阻止浏览器默认查找，`Esc` 关闭）。
- 录入关键词回车或点「检索」按钮，高亮全部匹配项并定位到当前项；上一个/下一个循环跳转，显示「当前/总数」计数。
- 无匹配时提示「无匹配信息」；切换笔记自动清除上一篇的检索。

## 八、其它修复与优化
- 点击安装后真正退出 App 进程（修复 macOS 关闭窗口但进程仍在）。
- 历史去重、防止快速重复点击「生成历史」导致重复记录。
- 修复多处图片丢失（切换文档/重开/复制粘贴）、PDF/Word 导出异常、表格中文输入光标跳动等问题。
