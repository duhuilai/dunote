# v0.1.36

## 本地文件笔记图片不显示（关键修复）
- **根因**：从文件夹导入的本地文件笔记，内容存于内存 `localNotes`；切换/再次打开笔记时，编辑器用内存里的旧 `note.content` 去 `setContent`，把已经写入磁盘的最新内容（含图片）覆盖掉、没有显示出来——磁盘上其实有图片，只是没读出来。
- **修复**：本地文件笔记在「首次打开」和「切换进入」时，改为**以磁盘文件为权威重新读取最新内容**（`readTextFile` 该笔记的 `filePath`），不再依赖内存 `localNotes` 的旧值。写磁盘 + 内存同步仍保留为双保险，切回时图片正确显示。
