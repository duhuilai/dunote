# v0.1.41

## 修复：智能表格列设置菜单放到最外层
- **问题**：列设置菜单虽然已改为 `position: fixed`，但仍挂载在表格组件内部，可能被其它层叠上下文或父容器影响。
- **修复**：使用 `createPortal` 将列设置菜单和选项编辑器渲染到 `document.body` 最外层，z-index 最高，完全脱离表格容器限制。

## 优化：人员管理入职日期改为中文日历
- 原生的 `<input type="date" />` 显示依赖系统语言，可能显示英文。
- 改为自定义中文日期选择器：月份/星期全中文显示，支持点选日期，也支持「今天」「清除」快捷操作。
- 新增可复用的 `src/components/ui/DateField.tsx` 组件，便于后续其它模块统一使用。
