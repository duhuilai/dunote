# duNote v0.1.25

## 新功能 · 智能表格（路线1 起步）
- 在笔记表格中支持「列字段类型」：文本 / 人员 / 日期 / 单选 / 数字（右键表格 → 智能表格区设置）。
- 按列排序：升序 / 降序（数字按数值、日期按时间、其余按中文排序）。
- 按列筛选：按该列内容「包含」匹配，一键清除筛选（非匹配行隐藏）。
- 列统计：实时显示「可见数据行」数，数字列自动给出合计。
- 字段类型随笔记内容（HTML）一起保存，兼容现有 Gitee 备份结构。

## 说明
- 排序/筛选暂不支持含合并单元格（colspan/rowspan）的表格，普通数据表可正常使用。
- 交互逻辑已通过前端构建，建议在安装包中实测排序/筛选/统计效果。
