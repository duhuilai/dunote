## duNote v0.1.16

### 优化
- **Gitee 备份目录结构优化**：远程目录完全镜像本地「打开的文件夹」层级，末级文件夹使用笔记名，里面按紧凑时间戳（如 `20260710182398284.json`）存放每次备份。示例：`dunote/会议纪要/2026-05-11亚林所/2026-07-10182398284.json`
- 备份文件内仍包含本地绝对路径、相对路径、应用版本等详细元信息
- 兼容 v0.1.15 及之前的 `dunote-history` 旧目录结构，拉取历史时自动合并去重

### 平台支持
- Windows (MSI / NSIS)
- macOS (DMG)
