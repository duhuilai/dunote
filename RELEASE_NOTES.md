# v0.1.31

## 修复 / 重做
- **Word 导出彻底重做**：由原来「HTML 改后缀 .doc」改为生成真正的 `.docx`（Office Open XML，借助 docx 库），解决此前一系列问题：
  - 顶部标题不再重复出现（旧方案额外拼了一个标题，与模板自带标题叠加）。
  - 图片完整内嵌：插入时自动内嵌的 base64 图片会作为真实图片写入 docx；旧笔记里的网络图片也会尝试拉取后内嵌，拉取失败则保留「[图片]」占位，不再整篇空白。
  - 有序 / 无序 / 任务列表保留正确的层级与编号（使用 Word 原生多级编号，缩进清晰，含 1. / a. / i. 大纲样式）。
  - 字号、字体、颜色、高亮精确还原；正文默认中文字体。
  - 标准表格与智能表格（data-table）均转换为带边框的 Word 表格。
- 同步修复 HTML / Markdown / PDF 导出里「标题重复」的问题（内容已以 H1 开头时不再额外拼标题）。
- 导出文件名扩展名由 `.doc` 改为 `.docx`。
